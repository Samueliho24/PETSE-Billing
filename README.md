# PETSE-Billing

##Configuracion inicial

1 - Creacion de las carpetas FrondEnd y BackEnd

2 - Configuracion del Backend
    2.1 - Creacion de un entorno virtual (python -m venv venv)
    2.2 - Activacion del entorno (venv\Scripts\activate)
    2.3 -Instalacion de librerias (pip install fastapi uvicorn sqlalchemy pydantic python-multipart)

3 - Configuracion del FrondENd
    3.1 - Creacion del proyecto de React con VITE (npm create vite@latest frontend -- --template react)
    3.2 - Instalacion de las dependencias de React (npm install; npm install axios react-router-dom lucide-react antd)

4 - Configuracion de electron
    4.1 - Instalacion de los paquetes npm (npm install -y)
    4.2 - Instalacion de Electron (npm install electron electron-is-dev --save-dev)
